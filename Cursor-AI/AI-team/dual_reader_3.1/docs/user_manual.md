## Introduction

### What is Dual Reader 3.1?

Dual Reader 3.1 is a cross-platform ebook reader application that displays original and translated text side-by-side. It's designed to help you:

- Read ebooks in languages you're learning
- Understand content in unfamiliar languages
- Compare original and translated text simultaneously
- Learn languages through reading

### Key Features

- **Dual-Panel Display**: Original and translated text side-by-side
- **Smart Pagination**: Automatic page calculation
- **Translation Support**: 50+ languages
- **Offline Reading**: Books and cached translations work offline
- **Customization**: Themes, fonts, layouts
- **Progress Tracking**: Automatic reading progress
- **Bookmarks**: Save important pages
- **Chapter Navigation**: Quick chapter access

## Library Management

### Importing Books

**Method 1: Import Button**
1. Tap the floating "Import Book" button
2. Select an EPUB or MOBI file
3. Book is added to library automatically

**Method 2: Drag and Drop (Web)**
1. Drag an EPUB or MOBI file onto the library screen
2. Release to import
3. Book appears in library

**Supported Formats:**
- EPUB (.epub)
- MOBI (.mobi)

### Organizing Your Library

**Search:**
- Use the search bar at the top
- Search by title or author
- Results update as you type

**Sorting:**
- **Recent**: Most recently opened first
- **Title**: Alphabetical by title
- **Author**: Alphabetical by author
- **Progress**: By reading completion percentage

**Viewing Book Information:**
- Tap a book to see:
  - Cover image
  - Title and author
  - Reading progress
  - Last read date

### Managing Books

**Delete a Book:**
1. Long-press the book card (or use delete option)
2. Confirm deletion
3. Book is removed from library

**Note:** Deletion cannot be undone. The book file is removed from your device.

## Translation Features

### Setting Up Translation

1. Go to **Settings** â†’ **Translation**
2. Select **Translation Language** from dropdown
3. Enable **Auto-translate** (recommended)
4. Enable **Synchronized Scrolling** (optional)

### How Translation Works

**Process:**
1. Book content is sent to translation service
2. Translation is returned and displayed
3. Translation is cached locally
4. Cached translations used for offline reading

**First Translation:**
- Requires internet connection
- May take a few seconds
- Shows progress indicator

**Subsequent Translations:**
- Uses cached translations when available
- Faster loading
- Works offline

### Translation Settings

**Auto-translate:**
- Automatically translates pages as you read
- Disable to translate manually (future feature)

**Synchronized Scrolling:**
- Scrolls both panels together
- Keeps original and translation aligned
- Disable for independent scrolling

### Supported Languages

**Major Languages:**
- Spanish, French, German, Italian, Portuguese
- Russian, Chinese, Japanese, Korean
- Arabic, Hindi, Turkish, Polish, Dutch
- And 40+ more languages

**Language Selection:**
- Choose from 50+ languages in Settings
- Language codes shown in dropdown
- Full language names displayed

## Troubleshooting

### Common Issues

**Books Won't Import:**
- Check file format (EPUB/MOBI only)
- Verify file isn't corrupted
- Ensure sufficient storage
- Try different book

**Translation Not Working:**
- Check internet connection
- Verify language is selected
- Try different language
- Restart app

**Pages Not Displaying:**
- Adjust font size/margins
- Try different theme
- Restart app
- Re-import book

**App is Slow:**
- Large books take time initially
- First translation is slower
- Reduce font size
- Close other apps

**Settings Not Saving:**
- Check storage permissions
- Restart app
- Verify storage space
- Re-import settings
