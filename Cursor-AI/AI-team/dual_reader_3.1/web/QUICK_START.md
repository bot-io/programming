### 1. Generate Icons (Required for PWA)

Choose one method:

**Option A: PowerShell (Windows)**
```powershell
cd web/icons
.\create_placeholder_icons.ps1
```

**Option B: Python**
```bash
cd web/icons
pip install Pillow  # One-time setup
python create_placeholder_icons.py
```

**Option C: HTML Generator (Browser)**
1. Open `web/icons/generate_icons_simple.html` in your browser
2. Click "Generate All Icons"
3. Save downloaded icons to `web/icons/`

### 2. Build for Web

```bash
flutter build web --release
```

### 3. Verify Configuration

```bash
dart run web/verify_web_config.dart
```

### 4. Test Locally

```bash
flutter run -d chrome
```

Or serve the build output:
```bash
cd build/web
python -m http.server 8000
# Or use any static file server
```

**Icons Missing?**
- Run icon generation script (see step 1 above)

**Service Worker Not Registering?**
- Ensure HTTPS (or localhost)
- Clear browser cache
- Check browser console for errors

**PWA Not Installable?**
- Verify all icons exist
- Check manifest.json is valid
- Ensure service worker is registered
- Verify HTTPS is enabled
