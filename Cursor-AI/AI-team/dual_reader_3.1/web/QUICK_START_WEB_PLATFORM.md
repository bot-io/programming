## üöÄ Quick Start

### 1. Generate Icons (Required)

Choose one method:

**PowerShell (Windows - Recommended):**
```powershell
cd web
powershell -ExecutionPolicy Bypass -File generate_icons_dotnet.ps1
```

**Node.js:**
```bash
cd web
npm install canvas
node generate_icons_node.js
```

**Python:**
```bash
cd web/icons
pip install Pillow
python create_placeholder_icons.py
```

**Browser (No dependencies):**
- Open `web/icons/generate_icons_browser.html` in browser
- Click "Generate All Icons"
- Download and save to `web/icons/`

### 2. Build Web App

```bash
flutter build web --release
```

### 3. Test Locally

```bash
cd build/web
python -m http.server 8000
# or
npx serve .
```

Open: http://localhost:8000

### 4. Verify Configuration

```bash
cd web
dart verify_web_platform_settings.dart
```

### 5. Deploy

Deploy `build/web/` to:
- **GitHub Pages:** Push to `gh-pages` branch
- **Netlify:** Connect repo, set build command: `flutter build web --release`
- **Vercel:** Connect repo, set output directory: `build/web`
- **Firebase:** `firebase deploy`
